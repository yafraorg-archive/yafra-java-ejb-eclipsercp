The frames

Server / Backend part

Operating System and Database System

The server consists of different components which can be seen as building blocks. It starts with an operating system like ubuntu server or windows server. Next key building block is the database system which can be any sort of RDBMS which knows SQL dialect and provides a JDBC interface.

Core server

The core server consists of apache cayenne. Beside the database server an application server exists like apache geronimo.

Client part

Web client

The web client is simply the browser, all standard compliant XHTML 1.0 browsers are supported. As we use on the server side apache wicket, the compliance is dependent on the apache wicket component.

Fat client with UI

This is the client based on Eclipse RCP. It uses an EJB client connector to connect to the EJB server.

Fat client

Fat clients can be used as batch jobs or one off tools, mainly without user interface.
